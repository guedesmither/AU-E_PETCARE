# COMO ADICIONAR ÁUDIO DE ÁGUA DE FUNDO

## 🎵 ÁUDIO AMBIENTAL DE ÁGUA

O som ambiente de água está configurado para tocar suavemente ao fundo do site!

### 📁 **Onde colocar o áudio:**
1. **Nome do arquivo:** `water-ambient.mp3` (ou `.ogg`)
2. **Pasta:** `audio/`
3. **Caminho completo:** `audio/water-ambient.mp3`

### 🎛️ **Configurações aplicadas:**
- **Volume baixo:** 20% (muito suave)
- **Loop infinito:** toca continuamente
- **Autoplay:** tenta iniciar automaticamente
- **Fallback:** inicia no primeiro clique do usuário

### 🌊 **Características ideais do áudio:**
- **Duração:** 2-5 minutos (em loop)
- **Formato:** MP3 ou OGG
- **Qualidade:** 128 kbps (equilíbrio qualidade/tamanho)
- **Conteúdo:** Som suave de água corrente, cachoeira ou mar calmo
- **Tamanho máximo:** 2MB para carregamento rápido

### 🎯 **Tipos de som recomendados:**
- **Água corrente suave** (rio/cachoeira)
- **Ondas calmas** do mar
- **Fonte de água**
- **Chuva leve** (se preferir)

### 📱 **Compatibilidade:**
- **Desktop:** Autoplay funciona na maioria dos navegadores
- **Mobile:** Inicia após primeira interação (clique/toque)
- **Navegadores modernos:** Suporte total para HTML5 Audio

### 🔧 **Se o áudio não tocar:**
1. **Verifique o nome:** `water-ambient.mp3` exatamente
2. **Verifique a pasta:** `audio/`
3. **Teste local:** Abra `audio/water-ambient.mp3` no navegador
4. **Clique na página:** Para ativar em mobile

### ⚡ **Otimização:**
- **Bitrate:** 128 kbps suficiente para áudio ambiente
- **Mono:** Pode usar mono para menor tamanho
- **Fade in/out:** Adicione transições suaves no arquivo

### 🎛️ **Controle do usuário:**
- **Volume fixo:** 20% (não sobrecarrega)
- **Sem controles visuais:** mantém design limpo
- **Loop contínuo:** experiência imersiva

### 🚫 **Importante:**
- **Arquivo pequeno:** Para não afetar carregamento
- **Sem direitos autorais:** Use áudio livre ou licenciado
- **Testado:** Confirme que funciona em diferentes navegadores

O som criará uma ambiente relaxante e profissional para os visitantes! 🌊🐾
