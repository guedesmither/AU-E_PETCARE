# COMO ADICIONAR TOUR VIRTUAL

## 🎬 SEÇÃO DE TOUR VIRTUAL

A seção de Tour Virtual foi estrategicamente posicionada entre Serviços e Agendar Visita!

### 📍 **Posicionamento estratégico:**
- **Depois de Serviços:** Cliente já conhece o que oferece
- **Antes de Agendar Visita:** Cliente pode explorar antes de decidir
- **Funil ideal:** Conhece → Explora → Decide → Visita

### 📁 **Arquivos necessários:**

#### 🎥 **Vídeo principal:**
- **Nome:** `virtual-tour.mp4` (ou `.webm`)
- **Pasta:** `videos/`
- **Duração:** 2-5 minutos
- **Resolução:** 1080x1920px (9:16 vertical)
- **Formato:** MP4 (H.264) + WebM para compatibilidade

#### 🖼️ **Poster (thumbnail):**
- **Nome:** `tour-poster.jpg`
- **Pasta:** `images/`
- **Tamanho:** 1080x1920px (9:16 vertical)
- **Formato:** JPG ou PNG
- **Conteúdo:** Imagem atraente do tour

#### 🎬 **Vídeo de fundo (opcional):**
- **Nome:** `tour-bg.mp4`
- **Pasta:** `videos/`
- **Duração:** 10-30 segundos em loop
- **Efeito:** Sutil, não atrapalhar conteúdo

### 🎯 **Conteúdo ideal do tour:**

#### 📋 **Roteiro sugerido (formato 9:16 vertical):**
1. **Entrada e recepção** (10-15s)
2. **Área de creche (pan vertical)** (30-45s)
3. **Playground e lazer** (30-45s)
4. **Áreas de descanso** (20-30s)
5. **Instalações do hotel** (30-45s)
6. **Dormitórios** (20-30s)
7. **Área de alimentação** (15-20s)
8. **Saúde e banho** (20-30s)
9. **Pátio externo** (30-45s)
10. **Despedida e contato** (10-15s)

#### 📱 **Vantagens do formato 9:16:**
- **Mobile-first:** Perfeito para smartphones
- **Stories/Reels:** Fácil compartilhamento
- **WhatsApp:** Ideal para envio direto
- **Instagram:** Formato nativo
- **TikTok:** Máximo engajamento

#### 🎙️ **Áudio (opcional):**
- **Narração:** Descrição dos espaços
- **Música:** Instrumental suave
- **Efeitos:** Sons ambiente pets felizes
- **Volume:** Equilibrado com narração

### 🎨 **Qualidade técnica:**
- **Bitrate vídeo:** 8-12 Mbps para Full HD
- **Frame rate:** 30 fps
- **Estabilização:** Sem tremidas
- **Iluminação:** Bem iluminado
- **Áudio:** Claro e sem ruídos

### 📱 **Funcionalidades implementadas:**

#### ✅ **Recursos do player:**
- **Controles padrão:** Play/pause/volume/barra
- **Tela cheia:** Botão dedicado
- **Poster:** Thumbnail antes de play
- **Responsivo:** Funciona em todos dispositivos
- **Compatibilidade:** MP4 + WebM

#### 🎯 **Cards informativos:**
- **Play Completo:** Instruções claras
- **Tela Cheia:** Como expandir
- **Áudio Guiado:** Se tem narração

#### 🔄 **CTAs estratégicos:**
- **Tour Virtual:** Exploração online
- **Visita Presencial:** Conversão offline

### ⚡ **Otimização:**

#### 📊 **Tamanho de arquivo:**
- **MP4 ideal:** Até 100MB
- **WebM alternativo:** Até 80MB
- **Poster:** Até 500KB
- **Tempo de carregamento:** <5s em 3G

#### 🌐 **Compatibilidade:**
- **Desktop:** Chrome, Firefox, Safari, Edge
- **Mobile:** iOS Safari, Chrome Mobile
- **Tablets:** Full funcionamento
- **Navegadores antigos:** Fallback mensagem

### 🚀 **Como implementar:**

1. **Carregue os arquivos:**
   ```
   videos/virtual-tour.mp4
   videos/virtual-tour.webm (opcional)
   images/tour-poster.jpg
   videos/tour-bg.mp4 (opcional)
   ```

2. **Teste local:**
   - Abra `videos/virtual-tour.mp4` no navegador
   - Verifique se carrega e toca corretamente

3. **Teste no site:**
   - Atualize a página (F5)
   - Clique em "Tour Virtual" no menu
   - Teste todos os controles

- **Taxa de reprodução:** >70%

- **Conversão para visita:** >15%
