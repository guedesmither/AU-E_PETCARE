# COMO ADICIONAR VÍDEO DE FUNDO NA SEÇÃO HERO

## 🎹 VÍDEO DA PRIMEIRA SEÇÃO (AZUL)

O vídeo já está configurado para aparecer de forma suave atrás dos elementos!

### 📁 **Onde colocar o vídeo:**
1. **Nome do arquivo:** `hero-bg.mp4`
2. **Pasta:** `videos/`
3. **Caminho completo:** `videos/hero-bg.mp4`

### 🎬 **Como o vídeo funciona:**
- **Reprodução automática** (autoplay)
- **Sem som** (muted) - necessário para autoplay
- **Em loop** infinito
- **Ocupa toda a seção** (100% width/height)
- **Opacidade suave** (30%) para não atrapalhar o texto
- **Zoom sutil** (scale 1.1) para preencher espaços
- **Transição suave** de 2 segundos

### 🎨 **Efeito visual:**
- **Vídeo aparece atrás** do logo, título e botões
- **Gradiente azul** sobreposto mantém a identidade visual
- **Elementos decorativos** (🐾 e 🦴) continuam animados
- **Texto permanece legível** com contraste garantido

### 📱 **Compatibilidade:**
- **Desktop:** Reprodução automática perfeita
- **Mobile:** Funciona com `playsinline`
- **Fallback:** Mensagem de erro se navegador não suportar

### 🎯 **Dicas para o vídeo ideal:**
- **Duração:** 10-30 segundos (em loop)
- **Resolução:** 1920x1080 ou maior
- **Formato:** MP4 (H.264)
- **Conteúdo:** Cenas felizes de pets, instalações, brincadeiras
- **Cores:** Cores que combinem com o azul da marca

### ⚡ **Otimização:**
- **Tamanho máximo:** 10MB para carregamento rápido
- **Bitrate:** 5-8 Mbps para qualidade boa
- **Frame rate:** 24-30 fps

### 🔧 **Se o vídeo não aparecer:**
1. **Verifique o nome:** `hero-bg.mp4` exatamente
2. **Verifique a pasta:** `videos/`
3. **Teste local:** Abra `videos/hero-bg.mp4` no navegador
4. **Limpe cache:** Ctrl+F5 na página

O vídeo criará um ambiente dinâmico e profissional para a primeira impressão! 🐾✨
